Tips

Color Keys

I've been asked recently how to change the keyframes' color, in the timeline, and I thought maybe the tip could help, so why not sharing it !

The color of the keys, as displayed in the timeline, can be read on an attribute, in a node, in your scene. When you create a keyframe in maya, that will create, in your scene, an "animCurve" node. Through the Node Editor, display the inputs of an object on which you have some keyframes to see (and select) (one of) the connected animCurve node(s). Double click on it to display it in the Attribute Editor, then, under the 'Anim Curve Attributes' part, scroll down to find an attribute called "Use Curve Color" (just under the QTableWidget). The way it works is very similar to the drawing override : you check the Use Curve Color option, then choose your color with the Curve Color attribute just under.

If you want to modify the color of every key in your scene, you can do it pretty easily with a few lines : 


from maya import cmds, mel
for x in cmds.ls(type='animCurve'):
    cmds.setAttr(x + '.useCurveColor', 1)
    cmds.setAttr(x + '.curveColor', 0, 0, 1)

You can also add a line or 2 to change the sise of your anim keys. It's easy and fast, so why not doing it ! (It is the equivalent of what you'd do via Preferences > Settings > Time Slider > Key tick size)


aPlayBackSliderPython = mel.eval('$tmpVar=$gPlayBackSlider')
cmds.timeControl(aPlayBackSliderPython, e=1, tickSize=10)

Finally, even though it's not really related, remember that you can create keyframes with a different color (green, by default), using : 


cmds.keyframe(tds=1)

Mixing this tip with a script job, for instance, you can get a different color by default for every key of your animation. The limit that I see, for now, is that you can't have different colors for the different keys in the same curve. If this is one of your needs, feel free to tell it in the comments, so that I could share a script that I've done and which gives much more flexibility and control (PySide)
